# Support Ticket Intake Portal (Phase 1 Sample)

This is a simple React + Vite project sample for testing Learnlytica Phase 1 Project Evaluations.

## What it demonstrates
- React/Vite ZIP structure detection (`package.json`, `vite.config.js`, `index.html`, `src/main.jsx`)
- UI business flow:
  - Create ticket form
  - Validation
  - Success message
  - Recent tickets list

## Local run (optional)
```bash
npm install
npm run dev
```

## Upload target
Upload the generated ZIP (`phase1-support-ticket-intake-react-vite.zip`) in:
`/project-evaluations` -> `Reference Submission (Admin Validation)`
