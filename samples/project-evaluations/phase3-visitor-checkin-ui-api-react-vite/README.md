# Visitor Check-In Desk (Phase 3 Sample)

React + Vite sample project for Learnlytica Phase 3 (UI + API Integration).

Flow covered:
- visitor check-in form
- required validation
- success message
- today's visitors table updates
- API checks:
  - `GET /api/health`
  - `POST /api/visitors/check-in`
  - `GET /api/visitors/today`
